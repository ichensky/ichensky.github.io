gnuplot script `file.gp`
```sh
# set terminal wxt # Interactive terminal (default on many systems)
# set output 'data.png' # Output to a PNG file

# Use a 2D map view
set pm3d map 
set size square 

set palette gray

# Plot the data using the 'matrix' keyword
splot 'data.dat' matrix with image
```

Run 
```sh
gnuplot file.gp -p
```
